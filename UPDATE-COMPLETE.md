## 🚀 Deploy to GitHub

### Quick Steps:
1. Create new repo: "aeoess-website"
2. Upload ALL files from `aeoess-website` folder
3. Settings → Pages → Enable from main branch
4. Done! Live at: `yourusername.github.io/aeoess-website`

### Custom Domain (tymofii.me):
1. Add `CNAME` file with `tymofii.me`
2. Update DNS settings
3. Enable HTTPS in GitHub Pages
4. Live at: `tymofii.me`
